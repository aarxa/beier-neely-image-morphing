Image Morphing - Assignment Summary

Overview

This project implements the Beier-Neely image morphing algorithm, a feature-based warping technique that allows smooth transformation between two images.

Approach

The morphing process consists of three main steps:

  • Feature Line Interpolation: The keypoints in both images are interpolated based on a time step to create intermediate shapes.
  • Warping: Each image is deformed using the interpolated features to align with the intermediate structure.
  • Cross Dissolve: The warped images are blended together to create the final morphed frame.

Key Functions

  • BeierNeelyMorph: Interpolates line segments, warps images, and performs cross dissolve.
  • warp: Uses bilinear interpolation to adjust pixels based on feature lines.
  • bilinearInterpolate: Ensures smooth pixel transitions by averaging neighboring pixels.
  • CrossDissolve: Blends the warped images at each time step.
